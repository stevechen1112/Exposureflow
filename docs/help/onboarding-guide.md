# Onboarding Guide

歡迎使用 ExposureFlow。本指南協助您在無人工協助下完成基本啟用。

## 1. 建立 Workspace 與 Site

1. 登入後進入 **Onboarding**（`/app/{workspaceId}/onboarding`）
2. 若尚無 site，於 Settings 建立 site 並填寫網域
3. 確認 workspace 方案配額足夠（Settings → Billing）

## 2. 連接 Google Search Console

1. 前往 **Settings → Integrations**
2. 新增 GSC credential（OAuth 或 Service Account）
3. 點擊 **Sync GSC** 觸發首次同步
4. 回到 Onboarding 確認 GSC 步驟為完成

## 3. 查看 Exposure Opportunities

1. 同步完成後，進入 **Opportunities**
2. 依優先級檢視建議項目
3. 可篩選類型（content、technical、SERP 等）

## 4. 核准 Decision 與 Roadmap

1. 於 **Opportunities** 或 Decision 佇列核准 candidate
2. 在 **Roadmap** 檢視排程項目
3. Client workspace 可透過 Client Portal 參與核准

## 5. 產生第一份報表

1. 進入 Dashboard 或 Reporting
2. 選擇 site 與 delivery mode
3. 產生報表後可匯出 Markdown / PDF / DOCX

## 6. 團隊與權限

- **Settings → Members** 邀請成員
- 角色：owner、admin、strategist、editor、analyst、client_viewer
